What is Machine Learning?

Computer ko bina coding ke seekhne ki taqat dena hi Machine Learning hai. Hum machine ko examples dete hain aur wo khud patterns dhoondti hai.

Traditional

Data + Rules = Answers

Insan rules likhta hai.

Machine Learning

Data + Answers = Rules

Machine khud logic banati hai.

3 Types of Learning

🕵️

Unsupervised

🎮

Reinforcement
