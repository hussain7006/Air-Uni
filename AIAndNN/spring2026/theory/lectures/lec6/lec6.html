Types of Machine Learning

👨‍🏫 Guided

Supervised Learning

Yahan machine ko Labeled Data diya jata hai. Yani machine ko pehle se pata hota hai ke input kya hai aur uska "Correct Answer" kya hona chahiye.

Apple Image + Label "Apple"
Orange Image + Label "Orange"
Learning from Teacher
  • ✅ Classification (Spam vs No Spam)
  • ✅ Regression (House Price Prediction)
🕵️‍♂️ Independent

Unsupervised Learning

Yahan koi teacher nahi hota. Machine ko sirf Raw Data diya jata hai aur wo khud hi usme se hidden patterns ya groups (clusters) dhoondti hai.

No Labels
➡️
  • ✅ Clustering (Customer Segmentation)
  • ✅ Association (People who buy Milk also buy Bread)
🎮 Trial & Error

Reinforcement Learning

Ye "Inaam aur Saza" (Reward & Penalty) par chalta hai. Machine ek environment mein kaam karti hai aur har achi move par use Points milte hain.

🤖
🏃
🏆
Hit obstacle = -10 | Reach Goal = +100
  • ✅ Game AI (Chess, AlphaGo)
  • ✅ Robotics (Walking Robots)
