Section 03

Formula Breakdown

The two core Least Squares formulas — slope m and intercept c.

Slope m
// Least Squares slope formula
m =  [ n(Σxy) − (Σx)(Σy) ]  ← numerator
     ─────────────────────────
     [ n(Σx²) − (Σx)² ]  ← denominator
Intercept c
// Y-intercept formula
c =  [ Σy − m(Σx) ] ÷ n
⚠ Why not just use (y₂−y₁) / (x₂−x₁)?
The basic two-point slope formula works perfectly when you have exactly two points — it computes the slope of the unique line through them. But with multiple data points, no single line can pass through all of them simultaneously. Each pair gives a different slope estimate, so you would need to pick one arbitrarily.

The Least Squares method avoids this by finding the one line that minimises the sum of squared vertical errors — Σ(y⊂i; − ŷ⊂i;)² — across all points at once, giving an objectively best-fit model rather than an arbitrary pick.

Section 04

Step-by-Step Calculation

Substituting the summation values into each formula, one arithmetic step at a time.

Step 1 Calculate Slope (m)
m  = [ n·(Σxy) − (Σx)·(Σy) ] ÷ [ n·(Σx²) − (Σx)² ]
   // Substitute: n=4, Σxy=1060, Σx=20, Σy=180, Σx²=120
m  = [ 4 × 1060 − 20 × 180 ] ÷ [ 4 × 120 − 20² ]
m  = [ 4240 − 3600 ] ÷ [ 480 − 400 ]
m  = 640 ÷ 80
m = 8
Step 2 Calculate Intercept (c)
c  = [ Σy − m·(Σx) ] ÷ n
   // Substitute: Σy=180, m=8, Σx=20, n=4
c  = [ 180 − 8 × 20 ] ÷ 4
c  = [ 180 − 160 ] ÷ 4
c  = 20 ÷ 4
c = 5
Step 3 Write the Regression Equation
Substitute m = 8 and c = 5 into  ŷ = mx + c
ŷ  =  8x  +  5
// Verify predictions against original data:
x=2  →  ŷ = 8(2)+5 = 21  (observed 20  ·  residual = +1)
x=4  →  ŷ = 8(4)+5 = 37  (observed 40  ·  residual = −3)
x=6  →  ŷ = 8(6)+5 = 53  (observed 50  ·  residual = +3)
x=8  →  ŷ = 8(8)+5 = 69  (observed 70  ·  residual = −1)
// Sum of residuals = 0 ✓ — errors balanced on both sides of the line.
