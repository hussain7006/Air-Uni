Derivation Flow

LMS Gradient Descent Update Rules

1

The Cost Function (MSE)

Pehle hum apna objective define karte hain: Total Squared Error ka average nikalna.

J(θ) = 1/2m ∑ (h(x) - y)²

2

Applying Chain Rule

Calculus ka rule: Power '2' baahar aayegi aur andar wali term ka derivative multiply hoga.

∂J/∂θ₀ = 1/2m · 2 ∑ (h(x) - y) · ∂/∂θ₀(h(x)-y)

Note: 2 se 2 cancel ho jayega!

3

The Final Update Rules

Ye hain wo final equations jo aap apne Python code mein likhenge.

For Intercept (θ₀)

θ₀ := θ₀ - α · 1/m ∑ (h(x) - y)

For Slope (θ₁)

θ₁ := θ₁ - α · 1/m ∑ (h(x) - y) · x

Python Implementation

# m = total data points
# alpha = learning rate

for i in range(iters):
    h_x = theta_0 + theta_1 * x
    
    # Step 3 Mapping:
    theta_0 -= alpha * (1/m) * sum(h_x - y)
    
    theta_1 -= alpha * (1/m) * sum((h_x - y) * x)
